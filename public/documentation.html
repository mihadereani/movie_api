myFlix API documentation

Business Logic URL HTTP Method Request body data format Response body data format
GET list of movies /movies GET None A JSON object holding list of ALL movies
GET data about a single movie by title /movies/:Title GET None

A JSON object holding data (description, genre, director, image URL, whether it’s featured or not) about a single movie - Example:

{
"Title": "The Godfather",
"Genre": { "Name": "Crime", "Description": "Crime films, in ...", },
"Director": { "Name": F"rancis Ford Coppola", "Bio": "Francis Ford Coppola is an American film ...", },
"Actors": [Marlon Brando, ...],
"ImagePath": "String",
"Featured": "True",
}

GET data about genre of a movie /genre/:genreName GET None

A JSON object holding data about a genre (description) by name/title (e.g., “Thriller”) - Example:

{
"Genre": {
"Name": "Crime"
"Description": "Crime films, in ...",
}

GET data about a director by name /directors/:directorName GET None

A JSON object holding data of a director - Example:

{
"Name": "Francis Ford Coppola",
"Bio": "Francis Ford Coppola is an American ...",
"Movies": [ Apocalypse Now, The Conversation, The Godfather ],
"Birth": "1939-04-07",
"Death": "1980-12-30T00:00:00.000Z"
}

CREATE new user /users POST

A JSON object holding data about new user - Example:

{
"Username": "Lojze Pretnar",
"Password": "fdhadhaer11ath",
"Email": "lojzepretnar@outlook.com",
"Birthday": "1983-12-30",
"FavoriteMovies": []
}

A JSON object holding data about new user, including ID - Example:

{
"Username": "Lojze Pretnar",
"Password": "fdhadhaer11ath",
"Email": "lojzepretnar@outlook.com",
"Birthday": "1983-12-30T00:00:00.000Z",
"FavoriteMovies": [],
"_id": "63470fbb08550d439fa6e5ea",
"__v": 0
}

UPDATE user info /users/:Username PUT

A JSON object holding user's updates - Example:

{
"Username": "Lojze Pretnar updated",
"Password": "updatedfdhadhaer11ath",
"Email": "updatedlojzepretnar@outlook.com",
"Birthday": "1983-12-29"
}

A JSON object holding updated user's data - Example:

{
"_id": "63470fbb08550d439fa6e5ea",
"Username": "Lojze Pretnar updated",
"Password": "updatedfdhadhaer11ath",
"Email": "updatedlojzepretnar@outlook.com",
"Birthday": "1983-12-29T00:00:00.000Z",
"FavoriteMovies": [],
"__v": 0
}

ADD a movie to users list of favorites /users/:Username/movies/:MovieID POST None A text message showing that a favorite movie has been added
REMOVE a movie from users list of favorites /users/:Username/movies/:MovieID DELETE None A text message showing that a movie has been removed
DELETE user /users/:Username DELETE None A text message showing that a user email has been removed
